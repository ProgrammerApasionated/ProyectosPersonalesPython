## 1. Personal Expense Analyzer

**Problem**  
Many people lack a clear understanding of how they spend their money and how their expenses are distributed over time, making financial planning difficult.

**Objective**  
Develop an application that allows users to:
- Record daily expenses
- Classify them by category (food, transport, leisure, etc.)
- Calculate statistics such as average, maximum, and minimum expenses
- Obtain category-based percentages
- Save and load data from files

---

## 2. Product Inventory Manager

**Problem**  
Small businesses or individual users need to manage products, prices, and stock without relying on complex or expensive tools.

**Objective**  
Create an inventory system that:
- Supports full CRUD operations for products
- Manages stock levels and pricing
- Generates automatic reports
- Persists data using files
- Analyzes most expensive, cheapest, or low-stock products

---

## 3. Academic Grading System

**Problem**  
Students need a simple way to record grades and analyze their academic performance over time.

**Objective**  
Implement a system that allows users to:
- Manage students and subjects
- Record grades
- Calculate averages per student and per subject
- Detect pass/fail streaks
- Obtain academic success percentages

---

## 4. Advanced Rainfall Analyzer

**Problem**  
Recording rainfall data is straightforward, but analyzing it properly (streaks, maximums, averages, or patterns) is more complex.

**Objective**  
Develop an application that:
- Records daily rainfall data
- Calculates averages, maximums, and minimums
- Detects rainy and dry-day streaks
- Generates complete climate statistics
- Stores data persistently
